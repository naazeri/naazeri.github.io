First, you need to install Python 3. If you've already done this, you can skip this step.

> You can download the latest version of Python for your operating system from this link: [Download Python](https://www.python.org/downloads/)

After installing Python, open a terminal (CMD, PowerShell, or Bash) and run the following command:

```bash
pip install pygame
```

Depending on your operating system, you might need to use **pip3** instead of **pip**.

If you run into installation issues, try using an older version of Python instead of the latest one. For example, if Python 3.11 doesn't work, try version 3.10.

Now, to ensure everything is set up correctly, run the following command to launch a sample game made with PyGame:

```bash
python -m pygame.examples.aliens
```

Depending on your operating system, you might need to use **python3** instead of **python**.

If the game runs successfully, congratulations! PyGame is now installed.
